# games

Collection of small games.

## TANKS

This branch contains a dependency-free C++17 terminal version of **TANKS**.
Protect the base marked `A`, destroy enemy tanks marked `E`, and clear waves for score bonuses.

### Controls

- `P` starts or restarts the game.
- `W`, `A`, `S`, `D` move your tank and aim its cannon.
- `F` fires a shell.
- `Q` quits.

### Build, test, and run

```sh
make
make test
make run
```
